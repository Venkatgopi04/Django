How to implement Exception handling in Django:
==============================
In Django we do have two types of Exception Handling which are as follows

->Predefine Exception
->User_Define Exception/Custom Exception


->Predefine Exception:
===============
An Exception which is generated by Django Framework itself then it is said to be
Predefine Exception


views.py:
=====
from django.shortcuts import render 
from application1.models import Products 
from django.http import HttpResponse
def Test_Case1(request):
    data=Products.objects.get(Pid=1010)
    return HttpResponse(data)
 


->User_Define Exception/Custom Exception:
============================
User_Define Exception/Custom Exception can be define as a if django developer  defining
as exeption using try and except block.


views.py:
=====
from django.shortcuts import render 
from application1.models import Products 
from django.http import HttpResponse
def Test_Case1(request):
    try:
        obj1=Products.objects.get(Pid=1001)
    except:
        return HttpResponse("<tt><h1>Dear User Data is not found for 1010 record ...</h1></tt>")
    return HttpResponse(obj1)
